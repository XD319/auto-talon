# Quickstart

1. Initialize workspace: `talon init --yes`
2. Run first task: `talon run "summarize this project"`
3. Open Personal Assistant workspace: `talon tui`
4. Open runtime Ops view: `talon ops` (`talon dashboard` is a compatibility alias)

Useful checks:

- `talon task list`
- `talon trace <task_id> --summary`
- `talon audit <task_id> --summary`
