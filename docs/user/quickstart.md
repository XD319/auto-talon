# Quickstart

1. Initialize workspace: `corepack pnpm dev init --yes`
2. Run first task: `corepack pnpm dev run "summarize this project"`
3. Open chat UI: `corepack pnpm dev tui`
4. Open dashboard: `corepack pnpm dev tui --mode dashboard`

Useful checks:

- `corepack pnpm dev task list`
- `corepack pnpm dev trace <task_id> --summary`
- `corepack pnpm dev audit <task_id> --summary`
