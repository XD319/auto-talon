# Install

## Requirements

- Node.js `>=22.5.0`
- Corepack enabled

## Quick Install

```bash
corepack enable
corepack pnpm install
corepack pnpm build
corepack pnpm dev init --yes
```

Or use scripts:

- Linux/macOS: `bash scripts/setup.sh`
- Windows PowerShell: `./scripts/setup.ps1`

## Verify

```bash
corepack pnpm dev version
corepack pnpm dev doctor
```
